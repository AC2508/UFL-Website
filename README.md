# UFL Website

`UFL Website` is a React/Spring app for StarCraft 2 tournaments.

## Installation

### Frontend
1. Install the [React](https://nodejs.org/) recommended version.
2. Run the following command lines in the project path:
    + `npm install`
    + `npm start`
